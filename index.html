Autonomous RC Car

As a part of my NC State class ECE 306 (Intro to Embedded Systems), I built a remote control line-following car. The car uses a Ti MSP430 micro-controller at its heart, with a secondary control board built to control the power, motors, LCD screen, and IoT functionality. The code for the car was written with C in IAR Embedded Workbench. I also wrote a simple Android application to send remote control commands to the car.
